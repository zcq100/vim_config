## KeyMap
    <F2>    一键退出
    <F4>    关闭当前文件
    <F5>    重新载入当前文件
    <F10>   显示或取消显示对齐线
    <F11>   取消搜索高亮
    <F12>   显示或取消显示右边的文件名窗口
    Ctrl+p  打开文件

## How to Install
    $ git clone https://github.com/zcq100/vim_config.git ~/.vim_config
    $ cat ~/.vim_config/install.sh
    $ bash ~/.vim_config/install.sh
